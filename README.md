# msuper-gui
解压anroid刷机包中的super镜像，并挂载查看！当前只做了ubuntu下的！

**操作说明**

**步骤 1: 选择 Super 镜像文件**  
点击“选择 Super 镜像文件”按钮或使用菜单栏的“文件 -> 打开”，选择要处理的 `super.img` 文件。

**步骤 2: 处理镜像**  
点击“处理镜像”按钮，程序会：  
- 使用 `simg2img` 转换稀疏镜像  
- 使用 `lpunpack` 解包出各个分区  
- 在下拉列表中显示可用分区

**步骤 3: 选择分区**  
从“选择分区”下拉列表中选择想要挂载的分区镜像。

**步骤 4: 挂载分区**  
点击“挂载所选分区”按钮：  
- 会弹出密码输入对话框  
- 输入管理员密码确认  
- 挂载成功后会自动打开文件管理器

**卸载分区**  
使用完毕后，点击“卸载分区”按钮卸载已挂载的分区。

**清除缓存**  
使用“工具 -> 清除缓存”可以清理：  
- 转换后的镜像文件（`*_ext4`）  
- 解包的分区文件  

> 注意：清除缓存前会检查是否有分区已挂载，如有需要先卸载。

**工作目录**  
- 镜像处理：`~/.local/images/`  
- 解包目录：`~/.local/images/unpacked/`  
- 挂载点：`~/.local/images/mount/`
